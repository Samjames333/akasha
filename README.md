# AKASHA — A Living Codex of the Inner Arts

A single-file, immersive website: ~21,000 words of the world's contemplative and esoteric
traditions, presented honestly, wrapped in a hand-written WebGL journey.

**No build step. No dependencies. No tracking. One HTML file.**

## What's inside

| File | What it is |
|---|---|
| `index.html` | The entire site — scrollytelling descent through 12 worlds, glass reader, THE PATTERNS gallery (6 interactive sacred geometries incl. an infinite-zoom Indra's Net), generative ambient audio, print-to-PDF binding |
| `akasha_codex.pdf` | The pre-bound 66-page book edition |
| `indras_net.html` | The standalone living sigil — recursive Metatron lattice, true infinite zoom, live Kuramoto coherence |
| `the-unmasking.md` | The book the journey arrives at — a short path of unmasking, in Voice and Echo |
| `the-register-rule.md` | The companion framework — first philosophy, method, ethics, open questions, retired claims |

## Run it

Open `index.html`. That's it. Scroll to descend; tap ◈ for the patterns; every jewel is a door.
The **∴ Strand** section binds the whole codex to PDF from any browser.

## The ethos

Every tradition is presented at its strongest **and** accompanied by an honest note — what
scholarship supports, what it disputes, which doors are closed and stay closed. The register
rule holds throughout: the poetry never borrows the lab's authority; the claims never hide in
the poem's immunity.

Copy it. Share it. Improve it. It was made to be given away.
